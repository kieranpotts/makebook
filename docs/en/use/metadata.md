# Metadata

The `metadata.yml` file provides metadata for your book. The YAML block must start and end with `---`.

```yml
---
title: My Book Title
author: Harry A Potter
rights: MIT License
language: en-GB
tags: [book, test, etc]
abstract: |
  A short summary of the book.
mainfont: DejaVu Sans
---
```

See the [Pandoc documentation](http://pandoc.org/MANUAL.html#extension-yaml_metadata_block) for a full list of available metadata keys.
