# Chapters

Each chapter is a single Markdown file in the `chapters` directory. The files will be joined automatically by name, so it is recommended to use numeric prefixes to set the order of the chapters:

```
chapters/01-introduction.md
chapters/02-installation.md
chapters/03-usage.md
chapters/04-references.md
```

Each chapter should begin with a level one heading:

```md
# Chapter Name

First paragraph of the chapter.
```
